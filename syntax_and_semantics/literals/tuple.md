# Tuple

A tuple is typically created with a tuple literal:

```ruby
tuple = {1, "hello", 'x'} # Tuple(Int32, String, Char)
tuple[0]                  #=> 1       (Int32)
tuple[1]                  #=> "hello" (String)
tuple[2]                  #=> 'x'     (Char)
```

To create an empty tuple use `Tuple.new`.
